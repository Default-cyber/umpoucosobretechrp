🌞 Projeto Solar Site: Análise e Simulação Fotovoltaica

Plataforma web interativa desenvolvida para facilitar a análise e simulação de sistemas de energia solar fotovoltaica. Utiliza Python (Flask), HTML5, CSS3, JavaScript e integrações com Google Maps.

Principais funcionalidades: mapeamento solar interativo, calculadora de área, simulador de painéis, análise financeira e ambiental.

🚀 Sistema de Análise e Geração de Relatórios de NFe

Aplicação em Python (CustomTkinter) para processamento automatizado de arquivos XML de NFe. Extrai dados fiscais e gera relatórios em Excel usando Pandas e BeautifulSoup.
